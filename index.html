HTML: Semantics

Tag Name Description
<header> Header Basic structure of a web page.
<nav> Navigation Basic structure of a web page.
<main> Main Basic structure of a web page.
<footer> Footer Basic structure of a web page.
<section> Section Defines elements in a document, such as chapters,
headings, or any other area of the document with the same theme.
<article> Article Holds content that makes sense on its own such as articles, blogs, comments, etc.
<aside> Aside Contains information that is related to the main
content, but not required in order to understand the dominant information.
<figure> Figure Encapsulates all types of media.
<figcaption> Figcaption Is used to describe the media.
<video> Video Video media file.
<embed> Embed Used to embed a media file
<audio> Audio Audio media file.

HTML: Structure

Tag Name Description
<!DCOTYPE html> Document type This lets the browser know what version of HTML to expect.
<html> HTML This element will contain all your HTML code
<head> Head Information about the web page, like the title, belongs within the head element.
<Title> Title This element will contain the title of the page which is shown in the tab of the user's browser.
<a> Anchor This element defines a hyperlink.
<!-- comment --> Comment This will prevent the computer from reading the text inside this element as code

HTML: Tables

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
